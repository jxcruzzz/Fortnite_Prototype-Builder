# Fortnite Prototype Builder

## Descripción

Proyecto desarrollado en TypeScript que implementa los patrones de diseño Prototype y Builder mediante un sistema de personalización de personajes de Fortnite.

## Patrones Utilizados

### Prototype
Permite clonar skins base sin necesidad de crearlas nuevamente.

Ejemplos:
- Aura
- Skull Trooper
- Superheroe

### Builder
Permite construir un Loadout personalizado agregando:

- Pico
- Mochila
- Planeador
- Gesto
- Arma

## Funcionamiento

1. Se crea una Skin base.
2. Se utiliza Prototype para clonarla.
3. Se utiliza Builder para personalizar el Loadout.
4. Se muestra el resultado final en consola.

## Tecnologías

- TypeScript
- Programación Orientada a Objetos
- Patrones de Diseño
